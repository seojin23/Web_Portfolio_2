기말고사 프로젝트

바로가기

개요

리스트, 스택, 트리 등의 자료구조를 웹 브라우저에서 시각적으로 학습할 수 있도록 돕는 도구입니다.

각 자료구조의 삽입, 삭제, 검색 등의 연산을 직접 수행하며 메모리 상태와 구조 변화를 실시간으로 확인할 수 있습니다.

핵심 기술

D3.js를 활용한 SVG 기반 동적 시각화 기술

메모리 블록 단위 자료구조 상태 실시간 렌더링 기술

모듈식 자료구조 클래스와 동적 스크립트 로딩 기술

중간고사 프로젝트

바로가기

개요

다양한 웹 프레임워크를 소개하기 위한 전시형 웹페이지입니다.

가장 인기있는 7가지 프레임워크를 선정하여 각각의 특징을 알기 쉽게 전시했습니다.

핵심 기술

각 프레임워크의 이름, 로고, 주요 특징을 한 눈에 살펴볼 수 있는 슬라이더 구성

프레임워크 사용 시 실질적인 코드 변화를 직관적으로 비교하기 쉬운 예시 코드 뷰어 구성

앵커 스크롤을 활용하여 상세 설명 자동 이동 및 최상단 이동 Top버튼 구성
